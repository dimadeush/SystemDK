----------------------------------------------------------------------
Установка системы построения сайтов SystemDK V3.0:
----------------------------------------------------------------------

Загрузите все файлы из скачанного Вами архива на сервер (кроме файла readme.txt).
Установите для папки includes/data и всех файлов в директории includes/data CHMOD 777.
Установите для директорий themes/systemdk/cache, themes/systemdk2/cache, themes/systemdk/templates_c, themes/systemdk2/templates_c  CHMOD 777. 
Установите для всех папок в директории userfiles/  и для папки ckeditor/Djenx.Explorer/connector/php/cache CHMOD 777.
Установите для конфигурационного файла модуля Новостей (modules/news/news_config.inc), модуля Гостевая книга (modules/guestbook/guestbook_config.inc), модуля Магазин (modules/shop/shop_config.inc)  CHMOD 777.
Далее откройте браузер и наберите в нем адрес своего сайта, затем следуйте инструкциям программы – инсталлятора.
После установки системы удалите папку install с сервера и зайдите в администраторскую (url вашего сайта/папка администраторской) для дальнейшей настройки Вашего сайта.
